# Google Antigravity: Research Findings for Unwind Brain Integration

## Overview

**Google Antigravity** is an AI-first IDE that marks the beginning of the "agent-first" era in software development. Released November 18, 2025 alongside Gemini 3, it represents Google's shift from "Search" to "Action."

## Key Capabilities

### 1. Agent-First Architecture
- AI agents are the **primary workers**, not assistants
- Humans become **architects**, not typists
- Multiple agents can run simultaneously across workspaces

### 2. Cross-Surface Control
- **Editor:** Tab autocompletion, natural language commands
- **Terminal:** Autonomous command execution
- **Browser:** Built-in Chrome for testing and interaction
- All three surfaces are synchronized for powerful workflows

### 3. Development Modes
- **Plan Mode:** For complex tasks - generates a "Plan Artifact" for review before coding
- **Fast Mode:** For quick edits - immediate execution

### 4. Key Features
- **Agent Manager:** Mission control for all active tasks
- **Artifacts Panel:** Paper trail of logs, code diffs, and plans
- **Browser Preview:** Built-in testing environment
- **User Feedback Integration:** Guide and refine agent work across surfaces

## Technical Requirements
- macOS: Monterey (12+), optimized for Apple Silicon
- Windows: 64-bit Windows 10/11
- Linux: Ubuntu 20.04+, Debian 10+, Fedora 36+

## Origin Story
- Acquired **Windsurf** (Codeium) for $2.4 billion
- Led by Sergey Brin's return to "Founder Mode"
- Built from ground up assuming AI agent is primary actor
