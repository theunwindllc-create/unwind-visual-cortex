## 1. Core Mandate: Closing the Loop

The Infinite Mirror Vision Cell's primary function is to **close the feedback loop** between the Unwind Brain's strategic output (The Blueprint) and the real-world execution (The Action). It ensures that the system's internal code is always aligned with the external, aspirational philosophy of **Invicta** and **Fearless Loop**.

## 3. The Audit Cycle: Generating the Discipline Score

The Mirror will execute a strategic audit at the end of every major phase (or upon user request).

### Step 3.1: Data Collection
1.  **Retrieve Blueprint:** Fetch the JSON Blueprint for the last completed project.
2.  **Retrieve Performance Data:** Fetch the actual performance metrics (views, engagement, conversion) from the Supabase MCP (once P3-T002 is complete).
3.  **Retrieve Execution History:** Fetch the session history and debriefs for the project's execution phase.

### Step 3.2: Discipline Score Calculation
The **Discipline Score (0-100)** is a weighted metric that measures the gap between **Intent** and **Execution**.

*   **Intent Compliance (50%):** Did the final output adhere to the Brand Analyst's guidelines, the Emotional Tone Expert's pacing, and the Sales Optimizer's CTA placement?
*   **Performance Delta (30%):** How did the actual performance compare to the Performance Prediction Model's (P3-T005) forecast?
*   **Philosophical Alignment (20%):** Is the copy and strategic narrative generated by the system aligned with the **Invicta** and **Fearless Loop** principles?

### Step 3.3: Growth Vector Generation
If the **Discipline Score is below 90**, the Mirror generates a **Growth Vector**—a high-priority task for the Vision Cell to integrate into the roadmap.

*   **Example Growth Vector:** "Implement Pacing Micro-Adjustment Module to correct for 15% Pacing Delta identified in last 3 projects."
