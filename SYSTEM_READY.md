## 🤖 How It Works

### Daily Cycle (6:00 AM)

1. **AI Agent Spawns** via scheduled task
2. **Loads Context** from Google Drive
   - Tracking database
   - Latest debrief
   - Coordination docs
3. **Gets Next Task** using tracking manager
4. **Works for 30-45 minutes** on ONE task
5. **Creates Debrief** with detailed handoff
6. **Uploads Everything** back to Google Drive
7. **Next Day:** Another agent picks up where this one left off

### Multi-Agent Coordination

```
Day 1: Agent A → Works on P1-T001 → Creates debrief
Day 2: Agent B → Reads debrief → Works on P1-T002 → Creates debrief
Day 3: Agent C → Reads debrief → Works on P1-T003 → Creates debrief
...
Week 8: System complete, fully automated video production!
```

### Perfect Context Handoff

Each agent knows exactly what happened before because:
- **Tracking database** = Single source of truth
- **Debriefs** = Detailed session summaries
- **Mandatory protocols** = No shortcuts allowed
